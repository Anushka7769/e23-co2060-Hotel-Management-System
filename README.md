# TourismHub LK

TourismHub LK is a tourism and hotel booking platform.

## Main Modules

1. Tourist Website
2. Partner / Property Website
3. Admin Website
4. Backend API
5. MySQL Database

## Main Features

- Tourist registration and login
- Partner registration and login
- Partner can register multiple properties
- Property registration needs admin approval
- Admin can approve or reject properties
- Tourists can only see approved properties
- Tourists can book rooms
- Notification system
